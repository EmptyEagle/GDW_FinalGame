# GDW_FinalGame
 
TITLE:

Laser Panic

DESCRIPTION:

Start your engine! Ride down an endless hall while dodging lasers, fire, and other dangers along the way. Close dodges will reward increased points, if you’re feeling risky.

CONTROLS:

[IN-GAME]

A/D & Left/Right Arrow - Left/right movement

Space/W/Up Arrow - Jump

Escape - Pause

[MENU]

W/S & Up/Down Arrow - Move selection

Space/Enter - Confirm
